### Crear un repositorio

```shell
git init 
```

### Crear una rama (branch)

```shell
git branch <name>
git checkout -b <name>
```

### Eliminar una rama (branch)

```shell
git branch -D <name>
```

### Activar o cambiar a una rama (branch)

```shell
git checkout <name>
```

### Preparar los cambios a guardar

```shell
git add <name>
git add -A 
git add .
git add *
```

### Guardar los cambios 

```shell
git commit

```
